[[Tools]]

### Description

Maestro is a Scala / Spark framework that offers abstractions and standards for users to build, deploy and monitor production policies. If Quark is the ideal tool for exploratory analysis of valuation models and curves to produce optimal policies, Maestro is the ideal tool for representing and deploying these policies.
